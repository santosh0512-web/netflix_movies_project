Introduction
This dataset is about Netflix Movies & TV Shows. Datasets have 12 columns with some null values. For the analysis of the dataset, Pandas, plotly.express, and Datetime libraries. I divided the analysis process into several parts for step-wise analysis and to find out trending questions on social media for Bollywood actors and actresses.
